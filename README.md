# SampleWebProject
## Introduction
Web application with simple CRUD operations and user roles. Made for educational purpose.

## Technologies
- HTML 5;
- CSS 3;
- jQuery 3.5.1;
- Bootstrap 4.6;
- Java 11;
- Spring Boot 2.2.4;
- MySQL 8.0.

## Features
- Login and Register;
- Create, Read, Update, Delete students;
- Edit user information;
- Change user role;
- Disable users;
